## How to Use This Code (Step-by-Step)

### Step 1: Create the HTML File
1. Open Notepad (Windows) or TextEdit (Mac)
2. Copy the code above
3. Paste it into the text editor
4. Go to File > Save As
5. Name it: `portfolio.html`
6. Choose location (Desktop is easiest)

### Step 2: Open in Browser
1. Right-click the file
2. Select "Open with" > "Chrome" (or your browser)
3. Your website is now live locally!

### Step 3: Customize Your Content
1. Replace "Alice Martin" with your name
2. Update project titles and descriptions
3. Replace `project1.jpg`, `project2.jpg`, etc. with your own images
4. Update the email in the footer

### Step 4: Publish Online (Free Options)
- **GitHub Pages**: Upload your `.html` file to GitHub
- **Netlify**: Drag & drop your file at netlify.com
- **Vercel**: Upload at vercel.com
- **Surge**: Use `surge` command line tool
