このコードの使い方（ステップバイステップ）

ステップ1：HTMLファイルを作成する

  1. メモ帳（Windows）またはテキストエディット（Mac）を開く
  2. 上記のコードをコピーする
  3. ファイル → 名前を付けて保存 に移動
  4. ファイル名：portfolio.htmlと命名
  5. デスクトップに保存する

ステップ2：ブラウザで開く

  1. デスクトップでファイルを探す
  2. ファイルを右クリック
  3. 「プログラムから開く」→「Chrome」（またはお使いのブラウザ）を選択
  4. あなたのウェブサイトがライブになりました！

ステップ3：コンテンツをカスタマイズする

  1. 「Alice Martin」をあなたの名前に置き換える
  2. プロジェクトのタイトルと説明を更新する
  3. 画像ファイル名を自分の画像に置き換える
  4. 色をあなたのブランドに合わせて変更する
  5. 連絡先メールアドレスを更新する

ステップ4：オンラインで公開する（無料）

  • GitHub Pages：.htmlファイルをGitHubにアップロード
  • Netlify：ファイルをnetlify.comにドラッグ&ドロップ
  • Vercel：vercel.comにアップロード
